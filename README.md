# Campus Ride - Hyper-Local Vehicle Sharing 🏍️🚗
**Solving mobility for 800+ hostellers through peer-to-peer sharing.**

Campus Ride is a premium, hyper-local Android application designed for university campuses. It establishes a direct connection between student vehicle owners and those in need of a ride, specifically optimized for search results within a 500-meter radius.

## 🎯 Our Mission
Our goal is to solve the "last mile" travel problem for hostellers. By enabling owners to rent out their idle vehicles for a fair price, we create a win-win ecosystem where owners earn and students travel with ease. We believe that every student should have access to mobility without the burden of ownership.

## ✨ Key Features
*   **Hyper-Local Discovery**: Algorithms prioritize showing vehicles closest to your current hostel (within 500m).
*   **Flexible Pricing**: Dynamic dual-pricing model allowing owners to set both hourly and daily rates, automatically calculating the best price for the customer.
*   **QR Verification**: Seamless and secure rental start and return process using unique QR codes for a frictionless handover.
*   **Premium UI**: A modern, Myntra-inspired dashboard designed for speed and clarity.
*   **Complete Lifecycle Management**: Track active rides with real-time timers and calculate final costs automatically.
*   **History & Earnings**: Dedicated owner and customer dashboards to track active rentals, borrowing history, and overall earnings.
*   **Vehicle Controls**: "Invisible" mode for owners to temporarily hide their vehicle for maintenance without deleting it.

## 🛠️ Tech Stack
*   **Language**: Java
*   **Database**: Firebase Firestore & Authentication
*   **Images**: Cloudinary API
*   **Barcode/QR Scanning**: ZXing Android Embedded
*   **Design**: Material Design 3, Google Fonts (Outfit)
